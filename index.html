rexposome R package

Under the framework of the HELIX project, the Bioinformatics Research Group in Epidemiology (BRGE) from the Barcelona Global Health Institute (ISGlobal) has developed the R package rexposome. This R package, that was raised under Bioconductor framework, was designed to incorporate, explore and analyse exposome data using R.

ISGlobal HELIX
R Bioconductor rexposome

Package Features

exposome data loading

rexposome implements read_exposome and load_exposme functions. The first one is in charge to load into R exposome data stored in raw file and to create an ExposomeSet. The second one creates an ExposomeSet from already existing data.frames.

exposome data management

rexposome implement the ExposomeSet class. It is an eSet based class used to store and manage exposome data in R under the framework of Biocondcutor.

exposome data exploration

Several functions are implemented in rexposome to explore and describe exposome data. The methods tableMissings and plotMissings allow to analyse the amount of missing data. plotHistogram and plotFamily allow to see the behaviour of thee exposures. correlation and plotCorrelation helps to understand the relation between exposures.

exposome data analysis

rexposome allows to perform EXposome-Wide Association Study (EXWAS) in both univariate and multivariate ways using standard single-exposure association test or through an elastic net approach.
